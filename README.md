# SkyGuide - Personal Weather Assistant

Aplikasi panduan cuaca personal yang dibangun menggunakan fluter untuk memenuhi tugas UAS Pemprograman Mobile. Aplikasi ini dirancang untuk memberikan informasi cuaca real-time dengan sentuhan personal.

## Blueprint panduan pengembangan (Progressive Develoment)

Aplikasi ini dikembangkan secara bertahap selama 5 minggu untuk memastikan kualitas kode dan orisinalitas pengerjaan.

### Minggu 1 : Fondasi & Identitas (Sedang Berjalan)

- [x] Inisialisasi Proyek Flutter.
- [x] Setup Repository GitHub.
- [x] Pembuatan UI Halaman Login Sederhana (Input Nama).
- [x] Implementasi skema warna aplikasi (Light Blue Theme).

### Minggu 2 : Navigasi & UI Dashboard

- [ ] Implementasi fungsi pndah halaman (Navigation).
- [ ] Desain antarmuka Dashboard Cuaca.
- [ ] Penambahan widge informasi suhu, lokasi, dan ikon cuaca dummy.

### Minggu 3 : Integrasi API & Data Real-Time

- [ ] Registrasi OpenWeatherMap API.
- [ ] Implementasi Logic Networking (HTTP Request).
- [ ] Sinkronisasi data cuaca asli berdasarkan lokasi user.

### Minggu 4 : Fitur Cerdas & Optimasi

- [ ] Penambahan fitur "Saran Aktivitas" berdasarkan kondisi cuaca.
- [ ] Fitur pencarian cuaca di berbagai kota.
- [ ] Refactoring kode dan pembersihkan bug.

### Minggu 5 : Finalisasi & Deployment

- [ ] Pembuatan ikon aplikasi (Branding).
- [ ] Build APK/App Bundle.
- [ ] Pendaftaran ke Google Play Console (Internal Testing).
